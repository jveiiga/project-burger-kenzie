## Instalação

    - Clonar repositorio
    $ git clone https://github.com/jveiiga/project-burguer-kenzie.git

    FRONT-END 📺
    
    $ cd project-burguer-kenzie

    - Instalação de dependências
    $ yarn install
    ou
    $ npm install

    - Rodar projeto
    $ yarn dev
    ou
    $ npm dev
